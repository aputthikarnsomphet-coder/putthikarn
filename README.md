## 📦 การติดตั้ง

### ข้อกำหนด
- Python 3.7+
- pip

### ขั้นตอน

1. Clone repository
```bash
git clone https://github.com/aputthikarnsomphet-coder/putthikarn.git
cd putthikarn
```

2. สร้าง virtual environment
```bash
python -m venv venv
```

3. Activate virtual environment

**Windows:**
```bash
venv\Scripts\activate
```

**Mac/Linux:**
```bash
source venv/bin/activate
```

4. ติดตั้ง dependencies
```bash
pip install -r requirements.txt
```

5. รันแอปพลิเคชัน
```bash
python app.py
```

6. เปิดเว็บเบราว์เซอร์ไปที่
```
http://localhost:5000
```
